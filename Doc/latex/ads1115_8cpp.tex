\doxysubsection{Detailed Description}
Sets the A\+D\+S1115 to the correct configurations and operates accordingly. 

The configuration options are outlined in the header file \mbox{\hyperlink{ads1115_8h}{ads1115.\+h}}, and the selected macros for this purpose set the A\+DC to continuous coversion mode, a data rate of 860 S\+PS, a volatge range of +-\/4.\+096V, single-\/ended input from A\+I\+N0, and disables the comparator.
