\doxysubsection{Detailed Description}
A\+D\+S1115 Bit\mbox{[}1-\/0\mbox{]}. 

These bits perform two functions. When set to \textquotesingle{}11\textquotesingle{}, they disable the comparator function and put the A\+L\+E\+R\+T/\+R\+DY pin into a high state. When set to any other value, they control the number of successive conversions exceeding the upper or lower thresholds required before asserting the A\+L\+E\+R\+T/\+R\+DY pin.

The D\+E\+F\+A\+U\+LT setting disables the comparator
