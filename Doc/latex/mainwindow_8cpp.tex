Uses the mainwindow class defined in mainwidnow.\+cpp and the \textbf{ Qt} toolkit to generate, update and control the Pong G\+UI according to incoming E\+MG signals. The \mbox{\hyperlink{class_main_window}{Main\+Window()}} function is used to instantiate the game window, and generate a \textbf{ Q\+Graphics\+Scene()} object in which to place the G\+UI items. The server socket and host ip address are bound to allow U\+DP communication.
