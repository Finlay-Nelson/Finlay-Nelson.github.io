Detailed Description

This class is responsible for the data communications and threading involved with conversion and processing of incoming data to the ADS1115.

The GPIOlisten class monitors the ALERT/RDY pin and triggers an interrupt each time an AD conversion is ready. The data is then processed before sleeping until the next interrupt.

void GPIOlisten::interrupt ( void  )

Executed in response to an interrupt, the function returns emits the readyread() signal with the measured voltage.

Uses the readsig() function of the ads1115 class to measure an input voltage before emitting the readyread() signal containing the voltage result. A counter keeps track of the number of conversions performed (uses of the interrupt() function, each time alerting the onset of conversion through the qDebug() slot.
