Detailed Description

Generates and updates the components of the Pong gmae's GUI window.

This class instantiates the game components (the GUI scene, paddles, ball and scoreboard), and updates them based on the incoming EMG signal through the Qt signal slot mechanism.

◆ datapros

void Window::datapros ( float  inval)
slot

Performs the data processing and filtering of the input signals.

"hp1" is an instance of a high pass Butterworth IIR filer, of order given by the input parameter. The time-smoothed power of the signal is calculated, added to the plots through the memmove() function, and saved. A 1 channel UDP is sent to test control of the game.

Parameters
invalThe IIR filter order for processing.
