\doxysubsection{Detailed Description}
This class is responsible for the data communications and threading involved with conversion and processing of incoming data to the A\+D\+S1115. 

The \mbox{\hyperlink{classGPIOlisten}{G\+P\+I\+Olisten}} class monitors the A\+L\+E\+R\+T/\+R\+DY pin and triggers an interrupt each time an AD conversion is ready. The data is then processed before sleeping until the next interrupt.
