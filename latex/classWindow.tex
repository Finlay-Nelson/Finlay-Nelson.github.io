\doxysubsection{Detailed Description}
Generates and updates the components of the Pong gmae\textquotesingle{}s G\+UI window. 

This class instantiates the game components (the G\+UI scene, paddles, ball and scoreboard), and updates them based on the incoming E\+MG signal through the Qt signal slot mechanism.
