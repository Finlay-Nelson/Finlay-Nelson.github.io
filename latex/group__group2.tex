\doxysubsection{Detailed Description}
The addresses of the registers of the A\+D\+S1115. 

The A\+D\+S1115 has four registers that are accessible via the I2C port. The Conversion register contains the result of the last conversion. The Config register allows the user the change the A\+D\+S1115 operating modes and query the status of the devices. The Lo\+\_\+thresh and Hi\+\_\+thresh registers set the threshold values used for the comparator function.
